# Neovim

Lazy.nvim 기반 설정.

## 링크

```bash
ln -sfn ~/workspaces/sharosoo/dotfile/nvim ~/.config/nvim
```

## 최초

```bash
nvim
# Lazy가 플러그인 설치 — 또는
nvim +Lazy sync
```

## 요구

Neovim **0.10+**. 시스템에 없으면 [scripts/install_neovim.sh](../scripts/install_neovim.sh) 참고 (수동 실행).
